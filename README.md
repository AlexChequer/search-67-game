# The Search for 67

Um jogo 2D desenvolvido em Unity como projeto da disciplina de Jogos Digitais do Insper.

## Como jogar
- **WASD / Setas** — Mover o personagem
- Colete todos os 8 67s dourados para vencer
- Evite o Tung Tung Tung Sahur — você tem 3 vidas
- Se o tempo acabar ou perder todas as vidas, é Game Over
- Controle Xbox/PS também suportado

## Mecânicas
- Cronômetro regressivo
- Sistema de vidas com invencibilidade temporária após tomar dano
- Inimigo que persegue o jogador
- Coletáveis em posições aleatórias a cada partida
